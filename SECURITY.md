# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please send an email to **furkan@furkanural.dev** with the following information:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours of your report
- **Status update**: Within 7 days with an initial assessment
- **Resolution target**: Within 90 days of the initial report

## Disclosure Process

1. You report the vulnerability privately via email
2. We acknowledge receipt and begin investigation
3. We develop and test a fix
4. We release a patched version
5. We publicly disclose the vulnerability after the fix is available

We appreciate your help in keeping Sayfa and its users safe.
